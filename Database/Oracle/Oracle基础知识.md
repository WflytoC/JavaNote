####Oracle常用工具

*  `Oracle Univeral Installer`：安装卸载Oracle数据管理系统

*  `Database Configuration Assistant`：创建/配置/删除/管理数据库

*  `Net Configuration Assistant`：监听器/命名方法/本地网络服务名/目录使用配置

*  `SQL Plus`：基于命令行

*  `SQL Developer`：图形化的Oracle操作平台
*  `PL/SQL Developer`：第三方工具
*  `TOAD`：第三方工具
